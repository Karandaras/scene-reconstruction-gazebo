RobotControllerPlugin Documentation

rcjoint Element (required)(1..n)

gname Element (required)(string)
    Name of Joint in the Model

fname Element (optional, defaults to gname)(string)
    Name of Joint in the stored data

offset Element (optional, defaults to 0) (double)
    Offset between Model and data



Example
<rcjoint>
	<gname>Name</gname>
	<fname>Another Name</fname>
	<offset>0.0</offset>
</rcjoint>
